# School Info Setup

Walk the user through documenting school information for each child. Ask which child to start with, then repeat.

## Per Child
Create an entry titled "[Child Name] — [School Name]" for each child with:

### School Details
- School name and address
- School phone number and website
- School hours (start/end, early release days)
- Bus number and pickup/dropoff times (if applicable)
- Carpool lane number or pickup procedure

### Key Contacts
- Principal name and email
- Homeroom/main teacher name and email
- School counselor name and email
- School nurse phone
- Front office phone and email

### Portal & Logins
- Student portal URL (e.g., PowerSchool, Canvas, Schoology)
- Student username and password
- Parent portal URL and login
- Lunch account number and balance check URL
- Library card number (if school-issued)

**Sensitive fields:** Portal passwords, lunch account number

### Activities & Sports
- Current extracurriculars (club name, day/time, coach/sponsor)
- Sports teams (sport, coach name/phone, practice schedule)
- Instrument/music (teacher, lesson day/time)
- Uniform sizes and where to order

### Important Dates
- School calendar link
- Conference dates
- Key deadlines (registration, forms, fees)

## Lunch & Meals
Create an entry titled "School Lunch Accounts" with:
- Per child: school, lunch account number, PIN, auto-reload setup
- Allergy alerts filed with school? (yes/no, what allergies)
- Lunch menu URL
