# Vehicle Info Setup

Walk the user through documenting each family vehicle. Ask how many vehicles they have, then create an entry for each one.

## Per Vehicle
Create an entry titled "[Year] [Make] [Model]" for each vehicle with:

### Basic Info
- Year, make, model, trim, color
- VIN (Vehicle Identification Number)
- License plate number and state
- Mileage (approximate current)

### Registration
- Registration expiration date
- Where registration card is kept
- DMV or renewal website

### Insurance
- Insurance company and agent name
- Policy number
- Coverage type (liability, full coverage, etc.)
- Monthly/annual premium
- Deductible amounts (collision, comprehensive)
- Claims phone number
- Roadside assistance phone number

### Loan / Payment (if applicable)
- Lender name and account number
- Monthly payment amount
- Payoff date
- Auto-pay setup? Which account?

### Maintenance
- Oil change interval and last oil change date/mileage
- Tire size and last rotation date
- Preferred mechanic/shop name and phone
- Dealership service department and phone
- Warranty details (bumper-to-bumper, powertrain, expiration)

**Sensitive fields:** VIN, policy number, loan account number
