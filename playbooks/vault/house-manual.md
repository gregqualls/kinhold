# House Manual Setup

Walk the user through building a complete house manual. Create a separate vault entry for each section below. Use the "House" category (create it first if it doesn't exist).

## WiFi & Network
Create an entry titled "WiFi & Home Network" with:
- WiFi network name (SSID) and password
- Guest network name and password (if any)
- Router brand/model and admin URL (usually 192.168.1.1)
- Router admin username and password
- ISP name, account number, and support phone number
- Modem location in the house

**Sensitive fields:** WiFi password, router admin password

## Alarm & Security
Create an entry titled "Alarm & Security System" with:
- Alarm company name and account number
- Alarm code(s) — who has what code
- Monitoring phone number
- Security camera system (brand, app name, login)
- Smart lock codes (front door, garage, etc.)
- Gate code (if applicable)

**Sensitive fields:** Alarm codes, smart lock codes, gate code

## Utilities
Create one entry titled "Utility Accounts" with a table or list:
- Electric: provider, account number, phone
- Gas: provider, account number, phone
- Water: provider, account number, phone
- Trash/recycling: provider, pickup days
- Internet: provider, account number, phone, plan speed

## HVAC & Water Heater
Create an entry titled "HVAC & Water Heater" with:
- HVAC brand, model, installation date
- Filter size and replacement schedule
- Thermostat type (Nest, Ecobee, etc.) and login
- Water heater brand, model, age
- Recommended maintenance schedule

## Appliances
Create an entry titled "Major Appliances" with:
- For each appliance (fridge, dishwasher, washer, dryer, oven, microwave):
  - Brand and model number
  - Serial number
  - Purchase date and warranty expiration
  - Where the manual is (or link)

## Contractors & Services
Create an entry titled "Contractors & Service Contacts" with:
- Plumber: name, phone, notes
- Electrician: name, phone, notes
- HVAC technician: name, phone, notes
- Handyman: name, phone, notes
- Lawn care / landscaper: name, phone, schedule
- Pest control: name, phone, schedule
- House cleaner: name, phone, schedule

## Paint Colors
Create an entry titled "Paint Colors" with:
- For each room: color name, brand, finish, color code
- Where leftover paint is stored

## HOA / Neighborhood
Create an entry titled "HOA & Neighborhood Info" (if applicable) with:
- HOA management company and contact
- Monthly dues and payment method
- Key rules (parking, trash, noise, exterior changes)
- Community amenities (pool, gym) and access info
- Neighborhood watch or security patrol contact
